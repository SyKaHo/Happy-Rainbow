About the Product

Lucky Rainbow is a smart rainbow toss game that combines the classic rainbow toss with advanced IR sensor technology.
It automatically detects the landing point of the coin and counts the score, providing real-time updates and rankings through a web application and server.

Key Features

  • IR sensor to detect the coin's landing point
  • Automatic score counting and updating
  • Real-time web application and server to display scores and rankings
